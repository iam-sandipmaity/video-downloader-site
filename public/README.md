# Public Assets

## Favicons & App Icons

✅ Created SVG-based favicon and app icons with teal/emerald gradient download icon

### Files:
- **favicon.svg** (32x32) - Main favicon, sharp at any size
- **logo192.svg** (192x192) - Apple touch icon and PWA icon
- **logo512.svg** (512x512) - High-resolution PWA icon

### Design:
- 📥 Download icon in white
- 🎨 Teal to emerald gradient background (#14B8A6 → #10B981)
- ⚡ SVG format for crisp rendering at any scale

### Browser Support:
- Modern browsers: Use SVG favicon
- Legacy browsers: Fallback to favicon.ico (if provided)
- iOS: Uses logo192.svg for home screen
- Android PWA: Uses logo192.svg and logo512.svg

## Folders:
- **images/** - App mockups and screenshots
- **apks/** - Android APK files for download

## Notes:
All icons follow the app's teal/emerald color scheme and feature a simple download arrow design that's instantly recognizable across all sizes.
